 - Записывает дату и время вызова,
 - Имя функции через `__name__`,
 - Все позиционные `(args)` и именованные `(kwargs)` аргументы,
 - Возвращаемое значение,
 - Добавляет запись в конец указанного файла (в режиме `'a'`).
